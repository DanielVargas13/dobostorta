Dobostorta
==========

The minimal browser.

## Install
```
$ git clone http://github.com/macrat/dobostorta.git
$ cd dobostorta
$ qmake
$ make
$ make install
```

## Usae
### Shortcuts
#### Scroll
|Key            |Description    |
|:--------------|:--------------|
|Ctrl-k or up   |scroll to up   |
|Ctrl-j or down |scroll to down |
|Ctrl-h or left |scroll to left |
|Ctrl-l or right|scroll to right|

#### History
|Key            |Description    |
|:--------------|:--------------|
|Ctrl-i         |forward history|
|Ctrl-o         |back history   |

#### Bar
|Key            |Description    |
|:--------------|:--------------|
|Ctrl-:         |focus to bar   |
